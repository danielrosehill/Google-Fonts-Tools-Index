# Google Fonts Tools Index [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

[![View on GitHub](https://img.shields.io/badge/View%20on-GitHub-181717?logo=github&logoColor=white)](https://github.com/danielrosehill/Google-Fonts-Tools-Index)

An index of utilities and libraries for working with Google Fonts — with an emphasis on installers for Linux desktops (CLI and GUI), plus developer libraries and the upstream font source itself.

## Scope

### In Scope
- CLI tools to download, install, and manage Google Fonts locally
- GUI applications for browsing and installing Google Fonts on Linux
- Developer libraries / helpers that integrate Google Fonts into apps and build pipelines
- The upstream Google Fonts source repository
- Claude Code / AI skills that interact with Google Fonts

### Out Of Scope
- Generic system font managers that don't specifically target Google Fonts
- Hosted webfont CDN services
- Font editors and typography design tools
